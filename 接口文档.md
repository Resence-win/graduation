# 八、文件上传说明

系统中的图片文件 **不会存储在数据库中**，数据库只保存图片访问地址。

图片上传流程：

```
前端上传图片
       ↓
SpringBoot 接收文件
       ↓
保存到服务器本地目录
       ↓
返回图片URL
       ↓
数据库保存URL
```

本系统图片存储目录：

```
/upload/merchant/
```

数据库示例：

```
logo = /upload/merchant/logo1.png
```

浏览器访问：

```
http://localhost:8080/upload/merchant/logo1.png
```

未来系统部署时可以替换为 **云对象存储（例如阿里云OSS）**。
