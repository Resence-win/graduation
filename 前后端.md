# 校园一卡通系统（Campus Card System）

## 一、项目结构

本项目采用前后端分离架构。

| 模块  | 项目名称               |
| --- | ------------------ |
| 前端  | campus-card-web    |
| 后端  | campus-card-server |
| 数据库 | openGauss          |

项目目录结构示例：

```text
campus-card
├── campus-card-web        # 前端项目
└── campus-card-server     # 后端项目
```
